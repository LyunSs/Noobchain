# Noobchain: A Simple Blockchain Implementation in Java

Noobchain is a basic blockchain implementation developed in Java. It follows the principles outlined in the tutorial "[Link Creating Your First Blockchain with Java. Part 1. ](https://medium.com/programmers-blockchain/create-simple-blockchain-java-tutorial-from-scratch-6eeed3cb03fa)". This project aims to provide a foundational understanding of blockchain technology through hands-on coding.
